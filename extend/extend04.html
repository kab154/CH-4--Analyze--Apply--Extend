Hexadecimal, RGB, and HSL color codes format:

When it comes to creating color there are 3 different ways to create them that can be used. Color can be created by using hex, rgb, hsl which are all ways to create more visually pleasing website. that can change font and background colors. The difference between is how precise you want to be. Hex is easier way to create color, using six hexadecimals numbers you can create any color but can be more restricted. Using RGB which represent Red, Green, Blue you can be more precise in making color and can even design your own. Hsl is a combination of both using a straighforward interpretation. Using hsl you can customize your own color and easier to use.

Preferred color code format:

Picking color code format is all about how much you want to design your own website. I preferred to use hsl because having the option to be more precise in what color i want to use and being easy to understand is a bonus in using HSL.
